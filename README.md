## Description

FinanceTracker is a modern web application for tracking personal finances.  
It allows users to manage income and expenses, categorize transactions, and visualize spending trends.

The app is built with an **offline-first approach**:

- **Database:** PGlite
- **Persistence:** IndexedDB
- **Migrations & schema:** Drizzle ORM

This makes the project fast, portable, and usable without a backend.

---

## Features

* Add income and expenses in seconds
* Edit and delete transactions anytime
* Organize everything with custom categories
* Track your balance and monthly summaries
* Visualize your spending with charts and analytics (Recharts)
* Offline-first experience — works without internet
* Recurring transactions (subscriptions, rent, salary, etc.)
* Import & export transactions as **CSV**
* Clean and safe forms with validation (React Hook Form + Zod)

---

## Tech Stack

* **Framework:** React 19 + TypeScript
* **Bundler:** Vite 6
* **Database:** PGlite (PostgreSQL in browser)
* **Persistence:** IndexedDB (`idb`)
* **ORM / Migrations:** Drizzle ORM
* **Styling:** Tailwind CSS v4
* **UI Components:** shadcn/ui
* **Charts:** Recharts
* **Forms & Validation:** React Hook Form + Zod
